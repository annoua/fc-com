# How to Install
## What you need
* node.js
* Visual Studio Code, Notepad++ or a similar tool
## What to do
* install node.js
* open shell/cmd in this folder
* enter **npx install** (notice the x!)
* use **npm start** to start the server (notice the m!)
* go to http://localhost:3030/
